# 创作生产线

创作生产线把已定位的知识转化为可审阅的内容产物。它只定义阶段边界，不公开具体题材策略、写作技巧或风格规则。

| 阶段 | 负责事项 | 交付物 |
| --- | --- | --- |
| 分析 | 明确目标、受众、证据与限制 | 需求摘要与资料清单 |
| 结构规划 | 组织主题、顺序、信息缺口和验收标准 | 大纲或结构骨架 |
| 草稿生成 | 依照确认的骨架形成内容 | 可审阅草稿 |
| 验收与返工 | 检查目标、事实、结构与表达 | 通过结论或返工去向 |

不得从原始资料直接跳到最终成稿。结构规划不得擅自改写已确认目标；验收只决定通过或返工方向，具体修改回到对应阶段完成。
