# 公开模板维护协议

此文件是 [`public-template/AGENTS.md`](public-template/AGENTS.md) 的说明入口。它不替换当前私有知识库的运行规则。

公开模板的 Agent 只负责：保护 raw 不变、维护 wiki 链接和来源、按索引查询、记录变更、以及对创作项目执行分阶段工作流。具体内容规则、写作技巧和私有方法论由使用者在自己的私有库中维护。

它的基本行为是：先定位任务，再读取当前环节所需内容；完成后交付一个可复用摘要，并决定是否更新 wiki、写入 outputs 或进入下一环。这样能避免“所有资料、所有规则、所有历史产物”同时进入一次对话造成的上下文膨胀。
